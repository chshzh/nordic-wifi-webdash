### Memory Notes

- **P1 — nRF7002DK FLASH at 97.9%**: Only ~22 KB free. No room to add the BLE provisioning stack (board overlay explicitly disables `CONFIG_ZEGO_WIFI_BLE_PROV=n` with comment "only ~23 KB available after HTTP server and web resources"). This is a known constraint, not a regression. Future feature additions on nRF7002DK require careful flash auditing.
- The nRF54LM20DK target (2 MB flash) is not affected by this constraint.
- Compressed web assets (`rodata` section) account for ~279 KB of flash.
